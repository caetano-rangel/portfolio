# portfolio

Versão 1 do meu proprio portfolio feito com Html, Css e Javascript para front-end e Python com metodos de Flask para back-end.

Essa aplicação usa como base o MVC (model, view, controller) sendo assim, arquivos com final .html devem ser colocados em uma pasta template e
arquivos com o final .css e .js devem ser colocados em uma pasta static para funcionamento da aplicação.
